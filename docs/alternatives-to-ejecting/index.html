Eject 的替代选择

Ejecting 允许你自定义任意内容，但是执行此命令后，你必须自行维护配置与脚本。如果你有许多类似的项目，这可能会提升维护难度。这种情况下，我们建议 fork react-scripts 和其余你所需要的包来代替 eject。这篇文章深入讨论了实现方法。你可以在这个问题中找到更多的讨论项。
