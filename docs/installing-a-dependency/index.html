安装依赖

该工具生成的项目包括依赖项 React 和 ReactDOM。它还包括一些被 Create React App 作为开发依赖项的脚本。你可以使用 npm 安装其他依赖项（例如 React Router）。

npm install --save react-router-dom

或者你可以使用 yarn 代替：

yarn add react-router-dom

不仅是 react-router-dom，其他库也适用于此。
